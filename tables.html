TAG DESCRIPTION
<table> Define a table within the HTML page
<tr> Add a row to the table
<td> Add data to the table horizontally
<tbody> Section off tables, including headings
<thead> Define the heading row in the table
<tfoot> Define the footer for the table, suitable for totals
<scope> Accepts "col" or "row" to label headings for readability within the HTML document
<colspan> Force table to span multiple columns
<rowspan> Force table to span multiple rows
